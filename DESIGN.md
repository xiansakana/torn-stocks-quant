# DESIGN.md

## 气质与意象
暗色交易终端风格，参考 Bloomberg Terminal / TradingView 暗色主题。信息密集但不杂乱，数据驱动，专业冷静。场景：深夜交易室，多屏显示器上跳动着实时行情与指标曲线。

## 配色方案
- 背景：深灰黑 `#0f1117`（主背景）、`#1a1d29`（卡片/面板）
- 前景文字：`#e1e4ea`（主文字）、`#8b8fa3`（次要文字）
- 涨（买入信号）：`#22c55e`（绿）
- 跌（卖出信号）：`#ef4444`（红）
- 强调色/品牌色：`#3b82f6`（蓝）— 用于交互元素、选中态、图表辅助线
- 边框：`#2a2d3a`
- 悬停/聚焦：`rgba(59,130,246,0.15)`

## 字体排版
- 数字/数据：JetBrains Mono（等宽，表格对齐）
- UI 文字：系统字体栈（Tailwind 默认 sans）
- 大数字（价格/收益率）加粗，小字说明轻量

## 页面结构
- **全局侧边栏**：导航（Dashboard / Strategy / Backtest），收缩为图标
- **Dashboard**：卡片式股票列表 + 策略信号摘要 + 市场热力图
- **Stock Detail**：上方 K 线图（占 60vh），下方指标面板（RSI / MACD / Bollinger），右侧策略信号 & 模拟持仓
- **Strategy**：策略参数配置 + 全市场信号表 + 信号历史
- **Backtest**：策略选择 → 参数 → 收益曲线 + 回测指标面板

## 动效与交互
- 数据刷新：数字变化用 `transition` 平滑过渡（200ms）
- 卡片悬停：轻微 `translateY(-2px)` + 边框高亮
- 加载态：骨架屏（shimmer），禁用旋转 spinner
- 图表交互：hover 十字线 + tooltip

## 设计禁忌
- 不使用浅色/白色背景作为主背景
- 不在数据表格中使用圆角过大的卡片（8px 即可）
- 不使用彩色渐变按钮，保持扁平
- 价格下跌不用绿色（中国以外惯例），统一用红色表示跌
- 不使用动画弹跳效果，保持沉稳专业
